### INSTALL NIKOLA ON WINDOWS 10 ###
####################################
(1) Open console in admin mode at desired path and install `virtualenv`.
$ py -m pip install virtualenv

(2) The best way to install Nikola is to use pip in the virtualenv.
$ virtualenv nikola
$ cd nikola

(3) Download `lxml` and `Pillow` compiled for your installed Python version and install them using `pip`.
$ pip install lxml-3.6.1-cp35-cp35m-win32.whl
$ pip install Pillow-3.3.0-cp35-cp35m-win32.whl

(4) Install Nikola extras
$ pip install --upgrade "Nikola[extras]"

(5) Create demo site
$ nikola version

$ nikola init --demo
$ cd demo
$ nikola build
$ nikola serve -a 127.0.0.1 -b
